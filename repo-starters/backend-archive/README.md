# backend-archive

백엔드 공부 기록, 개념 정리, 트러블슈팅, 실험 메모를 쌓는 레포.

## 목적

- 공부 흔적을 남기기
- 배운 개념을 다시 꺼내 보기 쉽게 정리하기
- 면접 전에 빠르게 복습하기
- 포트폴리오에서 학습 지속성을 보여주기

## 다룰 주제

- Java 기초와 객체지향
- JDBC와 SQL
- 트랜잭션
- Spring 핵심 개념
- JPA/MyBatis 비교
- HTTP, REST, 인증/인가
- CS 기초

## 추천 구조

```text
docs
 ├─ java
 ├─ database
 ├─ spring
 ├─ network
 ├─ cs
 └─ trouble-shooting
```

## 작성 원칙

- 개념 정의보다 내 언어로 다시 설명하기
- 코드 예시 짧게 넣기
- 실수한 포인트를 함께 기록하기
- 면접 답변처럼 요약 문장 추가하기
