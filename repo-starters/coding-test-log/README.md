# coding-test-log

백엔드 취업 준비용 알고리즘 풀이 기록 레포.

## 목적

- 문제 풀이 습관 만들기
- 풀이 과정을 기록해 사고력을 보여주기
- 자주 틀리는 유형을 모아 복습하기

## 언어

- Java

## 관리 방식

- 플랫폼별 폴더 분리
- 문제마다 풀이 코드와 회고를 함께 기록
- 시간복잡도와 핵심 아이디어를 항상 적기

## 추천 구조

```text
boj
programmers
notes
```

## 문제 기록 템플릿

- 문제 이름
- 링크
- 사용한 알고리즘
- 시간복잡도
- 처음 접근
- 최종 풀이
- 다시 풀 때 체크할 점
